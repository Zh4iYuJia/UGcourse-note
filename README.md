## 🛠️ 如何使用

### 1. 获取笔记
你可以直接在 GitHub 网页上浏览，或者将仓库克隆到本地：
```bash
git clone https://github.com/Zh4iYuJia/UGcourse-note.git
```

### 2. 编辑与预览
推荐使用以下工具来获得最佳的 Markdown 编辑和阅读体验：
- **[VS Code](https://code.visualstudio.com/)**: 配合插件使用体验更佳。
- **[Typora](https://typora.io/)**: 所见即所得的 Markdown 编辑器。
- **[Obsidian](https://obsidian.md/)**: 强大的知识库管理工具。

### 3. 导出为 PDF (打印/离线阅读)
如果你需要将笔记导出为 PDF 格式：

#### 使用 VS Code
1. 安装 **Markdown PDF** 插件 (`yzane.markdown-pdf`)。
2. 打开 `.md` 文件。
3. 右键点击编辑器区域，选择 `Markdown PDF: Export (pdf)`。

#### 使用 Typora
1. 打开文件。
2. 选择菜单栏的 `文件 (File)` -> `导出 (Export)` -> `PDF`。
